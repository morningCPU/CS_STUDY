# 元组

+ `tuple`
元组是不可变类型,不可修改
```
a = (1,2,3)
```

+ 运算和list一样

+ 元组中如果只有一个元素要加上一个逗号
```
a = (1,)
```

+ 打包和解包
```
a = 1,2,3
#相当于 a = (1,2,3)
i,j,k = a
#相当于 i = 1,j = 2,k = 3
```
注意解包时个数要对应，不然会报错
针对这个问题可以使用星号表达式 `*` 
但是要注意的是 *i 会是一个列表，并且解包时只能有一个
```
a = 1,2,3,4,5,6
i,j,*k = a
```
+ 解包操作对所有序列都有效
```
i,j,*k = range(1,7)
```

+ 交换变量的值
```
a , b = b , a
a , b , c = c , b , a
```
