### 🌟 二、安卓开发核心技术 - 四大组件应用矩阵
| 组件类型  | 核心功能           | 项目应用场景               | 学习重点     |
| --------- | ------------------ | -------------------------- | ------------ |
| Activity  | 界面展示与交互     | 主训练界面                 | 管理生命周期 |
| Service   | 后台任务执行、计时 | 服务提醒                   | 前台服务实现 |
| Broadcast | 系统事件监听       | 设备连接状态监控           | 动态注册机制 |
| Provider  | 数据共享           | 训练计划导出、URI 权限管理 |              |
