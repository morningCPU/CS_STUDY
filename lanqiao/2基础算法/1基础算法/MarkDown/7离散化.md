# 离散化

## 1.简要介绍
+   空间压缩
+   比如：将原数组的元素值作为下标来处理
+   离散化数组要求内部是有序的

***

## 2.离散化的实现方法
```
//测试代码
#include <bits\stdc++.h>
using namespace std;
int main()
{
    int n;
    cin >> n;
    vector<int> a;
    for (int i = 1; i <= n; ++i)
    {
        int x;
        cin >> x;
        a.push_back(x);
    }
    cout << "原内容为：\n";
    for (auto i : a)
    {
        cout << i << ' ';
    }
    cout << '\n';
    sort(a.begin(), a.end());
    a.erase(unique(a.begin(), a.end()), a.end());
    cout << "离散化后的内容为：\n";
    for (auto i : a)
    {
        cout << i << ' ';
    }
    cout << '\n';
    int pos;
    int x;
    cin >> x;
    pos = lower_bound(a.begin(), a.end(), x) - a.begin();
    cout << pos;
}
```
