<h1 style = "text-align:center">最值查找</h1>

备注：要包含头文件 `<algorithm>` 

## 1.最小值
返回最小值

+   两个数
`min(num1,num2)` O(1)

+   多个数
`min({num1,num2,num3})` O(n)

用大括号将要比较的数括起来
***

## 2.最大值
返回最大值

+   两个数
`max(num1,num2)` O(1)

+   多个数
`max({num1,num2,num3})` O(n)

用大括号将要比较的数括起来
***
## 3.min_element(st,ed)
返回最小值的地址

| 参数  |   解释   |
| :---: | :------: |
|  st   | 开始地址 |
|  ed   | 结束地址 |

***
## 4.max_element(st,ed)
返回最大值的地址

| 参数  |   解释   |
| :---: | :------: |
|  st   | 开始地址 |
|  ed   | 结束地址 |

***
## 5.nth_element(st,k,ed)
返回类型是void,会对原来的内容产生影响
将地址k摆上正确的内容，k前面的数都不大于k，k后面的数都不小于k
注意这三个参数都是迭代器或指针

| 参数  |           解释           |
| :---: | :----------------------: |
|  st   |         开始地址         |
|   k   | 从小到大排序后正确的位置 |
|  ed   |         结束地址         |

示例：`nth_element(d.begin(),d.begin()+3,d.end())`
