## 5. 线程安全的懒汉式
```java
public class Sington{
    private static Sington(){}
    private static volatile Sington sington = null;
    public static getInstance(){
        synchronized(Sington.class){
            if(sington == null)
                sington = new Sington();
        }
        return sington;
    }
}
```
