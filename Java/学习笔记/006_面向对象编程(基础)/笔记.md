**关键字 package、import**
+ package 包
用于指明该文件中定义的类、接口等结构所在的包
package 顶层包名.子包名;
对 pack1\pack2\PackageTest.java 有 package pack1.pack2;
不要取 java.xxx 的包名

**包的作用**
+ 划分项目层次
+ 管理大型软件
+ 解决类命名冲突
+ 控制访问权限

**import 导入**
使用定义在其他包中的java类
+ import 在 package 和 主体之间
+ 如果导入的类或接口是java.lang包下的，或者是当前包下的，则可以省略此import语句。
+ 如果已经导入java.a包下的类，那么如果需要使用a包的子包下的类的话，仍然需要导入
