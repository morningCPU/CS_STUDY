标识符

标识符命名规则
+ 包名：全部小写
+ 接口名，类名：单词首字母大写
+ 变量名，方法名：第二个单词开始首字母大写
+ 常量：全部大写，单词之间用 _ 连接

变量
数据类型，变量名，变量的值

Java中变量的数据类型分为两大类：
+ 基本数据类型 ： 整形，浮点型，字符型，布尔型
整形:byte,short,int,long
浮点型:float,double
字符型:char
布尔型:boolean
+ 引用数据类型：数组，类，接口，枚举，注释，记录
数组: []
类: class
接口: interface
枚举: enum
注释: annotation
记录: recode
